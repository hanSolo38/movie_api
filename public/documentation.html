Movie API Documentation

Returns movie information along with the genre details and director biographies.

Users can create a profile where they can add/delete their favorite movies.

Description Endpoint URL Method Type Query Parameters Request Body Format Response Format
Welcome message / GET N/A N/A String: "Welcome to My Movie App!"
List all movies /movies GET N/A N/A JSON array of movies and details
Get a movie by title /movies/:title GET title (string) N/A JSON object of the movie and details or an error message
Get genre description by genre name /movies/genre/:genreName GET genreName (string) N/A JSON object of the genre and description or an error message
Get director bio by director name /movies/directors/:directorName GET directorName (string) N/A JSON object of the director and bio or an error message
Create a new user /users POST N/A JSON: 
{
    "name": "string"
}
JSON object of the created user
Update user name by ID /users/:id PUT id (string) JSON: 
{
    "name": "string"
}
JSON object of the updated user or an error message
Add a movie to user favorites /users/:id/:movieTitle POST id (string), movieTitle (string) N/A Success message or error message
Remove a movie from user favorites /users/:id/:movieTitle DELETE id (string), movieTitle (string) N/A Success message or error message
Delete a user by ID /users/:id DELETE id (string) N/A Success message or error message
